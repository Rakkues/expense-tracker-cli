# Expense Tracker CLI
Small project from [roadmap.sh](https://roadmap.sh/projects/expense-tracker)

# Requirements
You need to have NodeJS installed in order to run this program

# How to Use
## Clone the Rep
```
git clone https://github.com/Rakkues/expense-tracker-cli.git
cd expense-tracker-cli
```

## Run using Node
```
node index.js <command>
```
You can list out the available commands by running
```
node index.js help
```
